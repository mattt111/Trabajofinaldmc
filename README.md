## 1. Descripción
Esta aplicación interactiva, desarrollada con **Streamlit**, integra los fundamentos de programación en Python para la gestión de actividades financieras. El proyecto permite desde el control básico de presupuestos hasta el uso de Programación Orientada a Objetos y funciones de orden superior para proyecciones de inversión.

## 2. Contenido de la Aplicación
La navegación se realiza mediante un menú lateral con los siguientes módulos:
 **Home**: Presentación del proyecto, datos del autor y tecnologías utilizadas.
 **Ejercicio 1**: Verificador de presupuesto simple utilizando **variables y condicionales**.
 **Ejercicio 2**: Registro de actividades financieras almacenadas en **listas y diccionarios**.
 **Ejercicio 3**: Cálculo de retorno esperado aplicando **programación funcional** .
 **Ejercicio 4**: Modelado de datos profesional mediante la **Clase Actividad** (**POO**).

## 3. Instrucciones de Ejecución
Para ejecutar este proyecto de forma local:
1. Descargar el repositorio
2. Verificar la version del streamlit en (requirements).
3.  Ejecutar la aplicación en cmd:  
    `streamlit run app.py`
